Instruções passo a passo do como iniciar o servidor do meu projeto

1 - Abra o CMD

2 - Busque o diretório Public do projeto com o comando CD
exemplo: cd C:\User\Documents\TCC\Internet_Web\Public

3 - agora que você está dentro da pasta public execute o comando "PHP -S localhost:8080" no terminal para iniciar o servidor na porta 8080, não necessáriamente você precise utilizar a porta 8080

4 - se você fez tudo certo digitando localhost:8080 em seu navegador você já vai conseguir abrir o meu projeto
